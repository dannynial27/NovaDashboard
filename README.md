# Nova - NextGen Finance Dashboard

![Nova Dashboard Preview](https://github.com/dannynial27/NovaDashboard/assets/YOUR_IMAGE_LINK_HERE)

A stunning, premium dark-themed financial portfolio dashboard designed with modern web aesthetics. Built to showcase fundamental web development skills with a high-end, futuristic "gempak" finish.

## 🚀 Features

*   **Glassmorphism Aesthetic**: Beautiful frosted glass panels and sleek semi-transparent UI elements.
*   **Vibrant Dark Mode**: Deep backgrounds accented with neon purples, teals, and pinks.
*   **Interactive Data Visualizations**: Smooth, responsive charting powered by Chart.js.
*   **Responsive Layout**: Fully adaptive design that looks great on mobile, tablet, and desktop using CSS Grid & Flexbox.
*   **Dynamic Micro-animations**: Engaging hover states and transitions that make the interface feel alive.
*   **No Build Tools Required**: Built with Vanilla HTML, CSS, and JS. Ready to run directly in the browser.

## 🛠️ Technology Stack

*   **HTML5**: Semantic and accessible markup.
*   **Vanilla CSS**: Custom design system using CSS variables, flexbox, and grid layouts (No Tailwind or external frameworks).
*   **Vanilla JavaScript**: DOM manipulation, event handling, and Chart.js initialization.
*   [Chart.js](https://www.chartjs.org/) (via CDN) for data visualization.
*   [Font Awesome](https://fontawesome.com/) for scalable vector icons.
*   [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter) for clean, modern typography.

## 💻 Running the Project Locally

Since this project has no build dependencies, setting it up is extremely simple:

1.  Clone the repository:
    ```bash
    git clone https://github.com/dannynial27/NovaDashboard.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd NovaDashboard
    ```
3.  Open `index.html` in your favorite web browser.
    *   *Tip: You can use an extension like VS Code Live Server for hot reloading while editing.*
